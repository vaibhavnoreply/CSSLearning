Unit Of Measure

PX, EM, and REM are primarily used for font sizing, %, VW, and VH are mostly used for margins, padding, spacing, and widths/heights.

Absolute

px: Unit is fixed and does not change based on any other element

Relative

em: Relative to the parent element font size, so if you wish to scale the element’s
                                        size based on its parent’s size, use EM.
rem: Relative to the root element (HTML) font size, so if you wish to scale the
                                        element’s size based on the root size, no matter what the parent size is, use REM
                                    
%: Relative to the parent element. is similar to VW and VH but it is not a length
                                        that is relative to viewport’s width or height.
vw: Relative to the viewport’s width. Vm is useful for creating full width elements
                                        (100%) that fill up the entire viewport’s width
vh: Relative to the viewport’s height. Vh is useful for creating full height
                                        elements (100%) that fill up the entire viewport’s height.
https://docs.elementor.com/article/595-difference-between-px-em-rem-percentage-vw-vh
BEM [Block Element Modifier]

It ensures that everyone who participates in the development of a website using proper naming conventions.

Block

                                Naming:.block

                                HTML:<div class="block">...</div>

                                CSS:.block { color: #042; }

Element

                                Naming:.block__elem 

                                HTML:<div class="block">...<span class="block__elem"> </span> </div> 

                                CSS:.block__elem { color: #042; }

Modifier

                                Naming:.block--mod or .block__elem--mod 

                                HTML:<div class="block block--mod">...</div> <div class="block block--size-bigblock--shadow-yes">...</div> 

                                CSS:.block__elem--mod { }
http://getbem.com/naming/
